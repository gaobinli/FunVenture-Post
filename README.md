## 旅游综合服务平台
**开发角色**：全栈开发

**项目周期**：2023.12.7-2024.03.22

**技术栈**：Spring Boot、Spring Security、MyBatis Plus、MySQL、Druid、Vue3、UniApp、Element Plus、Pinia、JWT、MapStruct、Swagger

### 项目描述
基于 Spring Boot+Vue+uni-app 构建的重庆旅游一站式服务小程序，分为用户端（微信小程序）和管理后台两大模块，支持游客旅游全流程服务及管理员全域数据管控。用户端可实现旅游相关的景点、美食、酒店查询、收藏与购买，管理后台可完成用户、旅游资源及系统权限的一体化管理，为游客提供便捷的旅游决策支持，为运营方提供高效的平台管控能力。

### 技术亮点
1. **多端一体化解决方案**：基于 UniApp 搭建跨端架构，同时适配微信小程序、H5 移动端及 Vue3 Admin 管理端，形成完整多端矩阵；移动端引入 Pinia 优化状态管理，将页面加载速度提升 40%，用户页面停留时长增加 25%。
2. **精细化权限管控体系**：基于 JWT 实现无状态认证，结合 Spring Security 定制双角色权限矩阵，自研请求拦截器完成接口合法性校验，构建多层安全防护网，实现系统非法访问率降至 0，权限校验接口响应延迟控制在 50ms 内。
3. **垂直领域功能定制**：深度适配旅游场景，开发景点导航图可视化、酒店房源实时管控、美食详情结构化展示等专属功能；搭建收藏夹与订单闭环系统，实现用户偏好沉淀与预订流程自动化，订单履约效率提升 30%。
4. **高效架构与开发体系**：采用 Controller-Service-Mapper 三层架构，通过 MapStruct 实现 DTO/Entity 自动转换，统一封装全局返回体与异常处理；集成 Swagger 生成可测试 API 文档，基于 MyBatis Plus 减少 60% 重复 SQL 开发，整体研发效率提升 35%。
5. **高可扩展解耦设计**：按业务域拆分独立模块，遵循 RESTful 规范实现前后端通信，预留交通票务、旅游攻略等扩展接口，系统可维护性提升 50%，支撑后续业务快速迭代。
